# 路*昊同学学习计划执行系统

部署地址：
- https://lzh20260211.pages.dev

## 本版目标
基于作业资料生成 20/30 天学习计划，而不是单纯电子化刷题。

## 已实现
- 自动生成 20/30 天学习计划（按学科轮转，每日任务上限可配）
- 每日任务勾选完成
- 学科学习计时（开始/停止，累计到当日任务）
- 拍照提交作业（本机保存，家长可查看）
- 家长端统计（总完成率、分学科完成情况）
- 手机/平板/电脑自适应
