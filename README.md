## 核心能力（按新需求）
- 20/30天学习计划自动生成（可配每日任务上限）
- 每日任务完成追踪（未完成任务顺延显示）
- 拍照提交作业（图片校验/大小限制/失败重试）
- 家长端查看（完成率、分学科进度、分学科累计时长）
- 计时（开始/暂停/继续/结束）与统计累计

## 数据结构（前端本地存储）
- `plan_v3`: 计划与任务状态
- `photo_v3`: 拍照打卡记录
- `time_v3`: 学科累计时长（ms）
